Tablas

Una tabla permite la distribución de la información organizada por filas (rows) y columnas (cols). Se utiliza la etiqueta html de nombree

Ejemplo:

Elemento 1 Elemento 2
Elemento 3 Elemento 4

Problemas...

Vamos a experimentar problemas cuando el número de columnas no sea el mismo en todas las filas. Ejemplo (lo siguiente está mal):

Elemento 1
Elemento 3 Elemento 4

Soluciones

Si deseamos que una celda ocupe más de una columna o fila, utilizaremos las propiedades colspan o rowspan. Ejemplo:

Elemento 1
Elemento 3 Elemento 4
